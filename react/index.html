  • What is React JS
A JavaScript library for building user interfaces


  • Created by Facebook
  • First release was in 2013
  • Current version - 17.0.2 (as of January 2022)
  • Huge projects on ReactJS:
    Facebook, Instagram, Netflix, New York Times, Yahoo! Mail, Dropbox, Walmart

Key features of React

  • JSX
JSX may remind you of a template language, but it comes with the full power of JavaScript.
  • Is not obligatory for React but is highly recommended to use
  • Comes from the fact that rendering logic (markup) is tightly connected with the UI logic
